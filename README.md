# Dudley Motimele – Web Developer Portfolio

## Overview
This is my personal portfolio website showcasing my skills, projects, and contact information.

- Parallax hero section with responsive background image
- About section with career summary
- Skills section (6 core skills, centered grid with icons)
- Projects section with screenshots and GitHub links
- Contact section with info + styled contact form
- Custom CSS for buttons, inputs, hover animations

- Responsive design principles
